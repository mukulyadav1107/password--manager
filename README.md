# 🔐 Password Manager (Python)

This project is a simple Password Manager built using Python and Tkinter.

## 🚀 Features
- Login system (Master Password)
- Add new passwords
- View saved passwords
- Data stored using file handling

## 🛠 Technologies Used
- Python
- Tkinter (GUI)
- File Handling

## 📂 How to Run
1. Open the project in VS Code
2. Run the Python file
3. Use the interface
